# redis-redisinsight

Provide offline memory analysis of Redis

## Prerequisites

- [docker](https://docs.docker.com/get-docker/)
- [docker-compose](https://docs.docker.com/compose/install/)

## Usage

Create Redis and RedisInsight

```bash
make up
```

Remove resources

```bash
make down
```
